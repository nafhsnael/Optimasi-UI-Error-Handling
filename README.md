# ✨ React Product Dashboard - Optimasi UI & Error Handling

Project ini merupakan tugas praktikum React untuk materi **Optimasi UI dan Error Handling** pada pertemuan 12 & 13.

Aplikasi dibuat menggunakan React dan berfokus pada peningkatan performa, efisiensi rendering, serta penanganan error agar aplikasi tetap stabil dan responsif saat digunakan.

---

## 📌 Fitur Utama

### 🛡 Error Handling
Menggunakan **Error Boundary** untuk menangani error pada komponen React sehingga aplikasi tidak langsung crash ketika terjadi kesalahan.

### ⚡ Optimasi Performa React
Project ini menerapkan beberapa teknik optimasi, antara lain:

- `React.memo`
  → Mengurangi render ulang komponen yang tidak diperlukan.

- `useMemo`
  → Menyimpan hasil komputasi agar tidak dihitung ulang setiap render.

- `useCallback`
  → Menjaga referensi fungsi tetap stabil.

### 🔄 Responsive Rendering
Menggunakan fitur React modern seperti:

- `Suspense`
- `useTransition`

agar tampilan tetap responsif ketika memproses data.

### 🪝 Custom Hooks
Logika aplikasi dipisahkan menggunakan custom hooks supaya kode lebih modular dan mudah dikelola.

---

## 🖥 Tampilan Aplikasi

Aplikasi menampilkan dashboard produk makeup dengan:
- Banner hero section
- Daftar produk
- Search & filtering
- UI yang responsif dan ringan

## 🚀 Menjalankan Project

### 1. Clone Repository

```bash
git clone https://github.com/nafhsnael/Optimasi-UI-Error-Handling.git
```

### 2. Masuk ke Folder Project

```bash
cd Optimasi-UI-Error-Handling
```

### 3. Install Dependency

```bash
npm install
```

### 4. Jalankan React App

```bash
npm start
```

Aplikasi akan berjalan di:

```txt
http://localhost:3000
```
